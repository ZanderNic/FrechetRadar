RadarDataGen: Synthetic Radar Point Clouds, Diffusion Models, and Fréchet Radar Distance (FRD)

Nicolas Zander

Thesis (PDF) Code (GitHub)

TL;DR

I introduce and validate the Fréchet Radar Distance (FRD), a domain-adapted, distribution-based metric inspired by FID. FRD replaces domain-specific feature extractors with random projections, enabling reproducible evaluation of sparse radar point clouds.

Context

This repository accompanies my Bachelor’s thesis conducted at Aumovio as part of the publicly funded nxtAIM project. The work focuses on distribution-based evaluation of generative models for radar data and on a task-independent similarity metric for synthetic radar point clouds.

Repository Overview

RadarDataGen is a research framework for generating parametrized pseudo-radar point clouds, discretizing them into structured grids, training U-Net-based diffusion models, and evaluating distributional similarity using FRD. Experiments are driven by JSON configs and use streaming statistics for efficient Gaussian estimation.

Abstract

The evaluation of synthetic radar data is challenging because established metrics such as FID rely on domain-specific feature extractors and do not transfer directly to radar. This work introduces and systematically evaluates the Fréchet Radar Distance (FRD), which uses random projections as a feature extraction mechanism and is independent of task-specific models.

To validate FRD, a controlled environment based on parametrized pseudo-radar generators is constructed, enabling targeted distribution shifts. As reference, an approximated log-likelihood is used to quantify similarity between the underlying data distributions. Results show FRD is consistent, reproducible, and correlates strongly with the approximated log-likelihood.

FRD is then applied to evaluate U-Net diffusion models trained on discretized radar grids. Under idealized training conditions, diffusion models can reproduce sparse occupancy and physically constrained structure. Model capacity analysis indicates diminishing returns beyond a certain size.

Method

Data Generation

Parametric pseudo-radar generators create controlled distribution shifts with primitives (lines, rectangles, circles) and clutter.

Discretization

Point clouds are mapped to fixed-size grids with presence indicators and per-cell offsets; collisions are handled via assignment.

Diffusion

U-Net diffusion models are trained on grids; sampling is evaluated against reference statistics.

FRD

Distributional similarity via Gaussian statistics in randomly projected feature spaces, computed with online mean/cov updates.
