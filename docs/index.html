RadarDataGen: Synthetic Radar Point Clouds, Diffusion Models, and Fréchet Radar Distance (FRD)

Nicolas Zander · OTH Regensburg

Bachelor’s Thesis

Evaluation of the Fréchet Radar Distance for the Assessment of Synthetic Radar Data

Thesis Code

Context

This repository accompanies my Bachelor's thesis conducted at Aumovio as part of the publicly funded nxtAIM project. The work focuses on the distribution-based evaluation of generative models for radar data and the development of a domain-adapted similarity metric for synthetic radar point clouds.

Repository Overview

RadarDataGen is a research-oriented framework for generating parametrized pseudo-radar point clouds, discretizing them into structured grid representations, training U-Net-based diffusion models, and evaluating distributional similarity using the Fréchet Radar Distance (FRD). The framework enables reproducible large-scale experiments via JSON-driven configurations and implements streaming statistics for efficient Gaussian estimation.

Abstract Thesis

The evaluation of synthetic radar data poses a central challenge in the field of generative modeling, as established quality metrics such as the Fréchet Inception Distance (FID) rely on domain-specific feature extractors and cannot be directly transferred to radar data. This work introduces and systematically evaluates the Fréchet Radar Distance (FRD), a distribution-based similarity metric inspired by FID. Instead of using pretrained domain-specific networks, FRD employs random projections as a feature extraction mechanism, making it independent of task-specific models. To validate FRD, a controlled experimental environment based on parametrized pseudo-radar generators is constructed. These generators allow targeted variation of statistical properties in synthetic radar point clouds. As a reference measure, an approximated log-likelihood is used to quantify the true similarity between the underlying data distributions. Experimental results demonstrate that FRD produces consistent, reproducible, and expected evaluations in controlled settings and exhibits strong correlation with the approximated log-likelihood. These findings indicate that FRD provides a robust measure for capturing structural similarities between radar data distributions. Building on this validation, FRD is further applied to evaluate U-Net-based diffusion models for synthetic radar data generation. Radar data is characterized by sparse occupancy, noise, and physically constrained structure, which fundamentally distinguishes it from image data. The results show that diffusion models, under idealized training conditions, are capable of reproducing these structural properties. Furthermore, the influence of model capacity is analyzed, revealing that beyond a certain model size no significant improvement in sample quality can be observed.

Method

Data Generation

Parametric pseudo-radar generators create controlled distribution shifts using geometric primitives (lines, rectangles, circles) and clutter.

Discretization

Point clouds are mapped to fixed-size grids with presence indicators and per-cell offsets; collisions are resolved via assignment.

Diffusion

U-Net diffusion models are trained on grids and evaluated by sampling and comparing against reference statistics.

FRD

FRD measures distributional similarity via Gaussian statistics in randomly projected feature spaces, computed with online mean/cov updates.
