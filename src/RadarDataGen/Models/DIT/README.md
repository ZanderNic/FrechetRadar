## Source and Attribution Notice
All Parts of the model implementation in this part of the repository are based on the publicly available DiT (Diffusion Transformer) code released by CompVis as part of the TReAD project. The original implementation can be found at:

https://github.com/CompVis/tread

This upstream code serves as a conceptual and structural basis for several components of the present implementation. In particular, the overall architecture of the DiT model, the design of the transformer blocks, the use of AdaLN modulation, the patchification and unpatchification logic, and the generation of two‑dimensional sinusoidal positional embeddings follow the ideas and structure of the original source.
